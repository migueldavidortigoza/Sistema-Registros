# Sistema de registros

Sistema desarrollado en JavaScript para registrar y gestionar datos, utilizando arrays de objetos y localStorage.

## Estado del proyecto
Vercion inicial(v1.0).
Proyecto base, funcional, con posibilidad de mejoras futuras.

## Funcionalidades
- Alta de registros
- Listado dinámico
- Persistencia con localStorage
- Carga automática al reacargar

## Tecnologías
- HTML
- CSS
- JavaScript
